# uuid-generator NPM

## A random unique ID generator npm package

### To Install

```npm
npm i @rameem2003/uuid-generator
```

#### To use in common js

```js
const uuid = require("@rameem2003/uuid-generator");

console.log(uuid()); // -> 'e8b7c5a2-8d32-4abc-90e4-bebc3d1a1234'
```
